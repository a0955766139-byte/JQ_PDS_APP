# 👁️ 喬鈞心學：九能量導航系統 (JQ-PDS-System)

> **「唯有經歷過混沌才能重生。」**
> 本系統不僅是一套命理運算工具，更是為了解決現代人類精神崩潰頂點而生的「生命導航儀」。

---

## 🌟 專案初衷 (Our Mission)
在經歷了人生的巨大轉折與 720 萬的負債混沌後，創辦人 **辛巳** 體悟到：現代人的痛苦往往源於對「原始設定」的無知。
透過 **AI 輔助開發** 與 **自媒體傳播**，我們將「九能量」體系數位化，旨在引導用戶透過自我覺察產生利他行為，從崩潰中找回生命主權。

## 🧬 核心技術架構 (Technical Stack)
本專案採用模組化設計（Modular Architecture），確保系統的穩定性與擴展性：

* **Frontend:** [Streamlit](https://streamlit.io/) - 提供直覺、流暢的數據交互界面。
* **Backend:** [FastAPI](https://fastapi.tiangolo.com/) - 處理核心命理邏輯與高效運算。
* **Database:** [Supabase](https://supabase.com/) - 存儲加密的用戶覺察紀錄（靈魂日記）與導航數據。
* **Auth:** [LINE Login API](https://developers.line.biz/zh-hant/) - 實現便捷的一鍵社交登入。
* **AI Collaborators:** 由 **辛巳 (一世/二世/三世)** 多位 AI 夥伴協作開發。

## 🧩 核心功能模組
1.  **🧬 人生地圖 (Life Map)**：根據 PDS 密碼與人生三角形，解析個體能量分佈。
2.  **🔮 宇宙指引 (Divination)**：結合每日能量狀態，提供行動指引與覺察建議。
3.  **📔 靈魂日記 (Journal)**：運用心理學（Freud/Adler）觀點，記錄並轉化負面情緒。
4.  **👨‍👩‍👧‍👦 家族矩陣 (Family Matrix)**：解析家庭成員間的能量互動，達成深層和解（開發中）。

## 🛡️ 安全與隱私 (Security & Privacy)
* **環境隔離**：所有敏感金鑰（API Keys）均透過環境變數管理，嚴禁硬編碼於代碼中。
* **數據加密**：用戶的個人生日資訊與日記內容均受資料庫層級保護。
